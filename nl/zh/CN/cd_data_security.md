# 保护数据    
{: #cd_data_security}  

{{site.data.keyword.contdelivery_full}} 在安全的高可用性环境中托管数据库：
   * 静态（GPFS、LUKS 和内置磁盘）和动态（HTTPS 和 SSH）加密数据。客户和系统凭证存储在加密磁盘上。
   * 针对高可用性配置应用程序和数据。
   * 对数据访问权的限制是仅限那些需要使用数据来支持和维护服务的用户。
