# 保護資料    
{: #cd_data_security}  

{{site.data.keyword.contdelivery_full}} 會在高度可用且安全的環境中管理您的資料庫：
   * 資料採取靜態加密（GPFS、LUKS 及內建磁碟）和進行中加密（HTTPS 及 SSH）。用戶端及系統認證儲存在已加密的磁碟上。
   * 配置應用程式及資料以獲得高可用性。
   * 只有需要資料以便支援及維護服務的使用者才能存取資料。
